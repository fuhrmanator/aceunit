AceUnit Future

The following features are planned for future versions:

  • Optional feature: Pluggable test runners
  • Optional feature: Create / finnish implementation of JUnit compatible loggers
  • Optional feature: Randomization of test order
  • Optional feature: Automatic generation of main() function
  • Optional feature: Off-Target Runner which runs the unit tests on-target by controlling a simulator, debug or other user-definable interface
  • Better logging of information, e.g. expected and actual values
  • Alternative test runner with more information, e.g. timing values
  • Optional feature: options to runner
    • Runtime configuration of loggers
    • Runtime configuration of suites, fixtures and test cases to execute.
